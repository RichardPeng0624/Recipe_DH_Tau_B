Welcome to the coding repository for exploring the atmosphere of the young super-Jupiter **DH Tau B** using **CRIRES+** data and modeling/retrieval workflows.

## Project Goals
- Build a reproducible analysis and retrieval workflow for DH Tau B.
- Process and inspect CRIRES+ spectra.
- Run atmospheric retrieval experiments and compare model assumptions.
- Track results, diagnostics, and configuration choices.

## Planned Structure
- `data/` raw and processed data pointers/metadata
- `notebooks/` exploratory analysis notebooks
- `src/` reusable analysis and retrieval code
- `configs/` retrieval and experiment configuration files
- `results/` outputs, plots, and tables

## Notes
Large raw data products should be stored outside git and referenced by path or download instructions.
